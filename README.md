# Service Install
## VERSION : V0.1_R20200325-03

## 1. Install Gulp
>$> npm install gulp-cli -g  
>...  
>$> npm install gulp --save-dev

## 2. Pull Repositories
>$> gulp updateSubmodule

## 3. Install npm libraries
>$> gulp installNPM

## 4. Set Databases
>$> gulp setDatabase



# Service Start

## Start PM2 
>$> gulp startPm2
***

## Check appliations 
>$> pm2 list
